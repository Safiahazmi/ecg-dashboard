Feature Unit Guide

0_pre-RR and 0_post-RR are RR interval timing features. They are displayed in milliseconds (ms).

0_qrs_interval is QRS duration. It is displayed in milliseconds (ms).

0_rPeak is the detected ECG peak amplitude from hardware. It is displayed as ADC/raw amplitude, not ms.

System Flow

1 AD8232 ECG sensor collects ECG signal.
2 ESP32 sends ECG features to Python bridge.
3 ML model predicts Normal or Abnormal.
4 PostgreSQL stores prediction data.
5 Dashboard displays result and history.
